## 🧱 System Build Philosophy (Cumulative)

This roadmap follows a strict cumulative architecture:

1. **Project 1 — ARC-Perceiver**  
   Classical ML feature extraction + rule templates to establish interpretable baselines.

2. **Project 2 — ARC-NeuralSolver**  
   CNN + attention + object-centric representations to improve transformation modeling.

3. **Project 3 — ARC-Programmer**  
   LLM-based DSL program synthesis with execution-guided filtering.

4. **Project 4 — ARC-Reasoner**  
   Chain-of-thought + GRPO + test-time search + self-correction for frontier performance.

Each project reuses and improves the previous layer rather than replacing it.

---

## 📚 Read → Build → Benchmark → Publish

Core operating loop for every phase:

1. **Read papers/books deeply**
2. **Implement from scratch (or near-scratch)**
3. **Benchmark on ARC-AGI**
4. **Write and publish findings (blog/workshop/paper)**

No phase is considered complete without measurable output.

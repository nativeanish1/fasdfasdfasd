## 🎯 Core Philosophy

1. **Read → Build → Benchmark → Publish.** Every book feeds a project. Every project produces a measurable ARC-AGI score improvement.
2. **Research-grade from day one.** No toy projects. Each project should be suitable for a workshop paper submission.
3. **Cumulative architecture.** Each project builds on the previous. By Project 4, you have a unified system.
4. **Honest feedback loops.** Track what works and what doesn't. Update this roadmap as you learn.

## ⚠️ Important Notes

- ARC-AGI-1 public evaluation set has 400 tasks. ARC-AGI-2 (2025) significantly increases difficulty.
- ARC prizes are actively awarded — this roadmap is structured to be competition-viable.
- All project target scores are *ranges*, not guarantees. They are calibrated to current SOTA at time of writing.
- This roadmap assumes ~4–6 hours of focused work per day, 5–6 days per week.
